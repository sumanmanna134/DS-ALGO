# Git Cheat sheet
## pull from main branch
```bash
git pull origin main
```
## fetch all branches
```bash
git fetch -all
```
## check status
```bash
git status
```
## add particular file in git
```bash
git add <file_name>
```
## add all modified files
```bash
git add -A
```
## commit changes
```bash
git commit -m "<message>"
```
## push in remote branch
```bash
git push origin <branch_name>
```
## discard all changes
```bash
git stash
```
